# AI-Powered Smart Retail Cart

An intelligent IoT-based shopping cart system that automates billing and enhances customer shopping experience using real-time spending intelligence.

## 🚀 Problem
Long checkout queues and lack of real-time budget awareness in retail stores lead to poor customer experience.

## 💡 Solution
A smart shopping cart powered by ESP32 and RFID that:
- Automatically detects items
- Calculates total bill in real time
- Predicts final spending
- Alerts users if budget is exceeded
- Provides intelligent shopping assistance

## 🏗 System Architecture
User → Smart Cart (ESP32 + RFID) → Processing Engine → Cloud/Database → Admin Dashboard

## 🛠 Tech Stack
- ESP32 (30-pin)
- RFID RC522
- I2C LCD Display
- Embedded C (Arduino IDE)
- WiFi Communication (Optional Cloud Integration)

## ✨ Key Features
- Automated billing system
- Budget alert mechanism
- Real-time cost tracking
- Scalable cloud-ready architecture
